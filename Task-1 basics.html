What is a browser?

A browser is an application program that provides a way to look at and interact with all the information on the World Wide Web.

Examples: Google Chrome, Firefox, Microsoft Edge etc..,..

What is internet?

A worldwide system of interconnected computer networks and electronic devices that communicate with each other using an established set of protocols.

Examples: Browsing websites, Sending emails, Engaging in social media, Streaming content, and Online shopping.

What is URL?

Uniform Resource Locator, is the web address that uniquely identifies a specific resource on the internet, such as a webpage, image, or document.

Examples: https://www.google.com/

What is domain

A human-readable internet address, like "google.com", used to identify and access websites.

Example: www.flipkart.com
-> www is a Sub domain.
-> flipkart is a Main domain.
-> .com is a Top-level-domain.

Difference between http and https?

HTTP and HTTPS are protocols that govern data transmission over the internet, specifically on the World Wide Web. While both transfer data, HTTPS is the secure version of HTTP, encrypting data to protect it from eavesdropping and tampering.

http: hypertext transfer protocol

https: hypertext transfer protocol secure

What is HTML?

HTML, which stands for HyperText Markup Language, is the foundation of most websites. It's a markup language used to structure and design web pages.
